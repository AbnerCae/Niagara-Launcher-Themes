# 📱 Niagara Launcher Themes

Bem-vindo ao repositório central de temas para o **Niagara Launcher**! Aqui compartilhamos criações da comunidade para deixar sua home screen mais bonita e funcional.

O objetivo deste projeto é criar uma biblioteca aberta onde qualquer pessoa possa baixar arquivos `.nlt` e aplicar instantaneamente, além de servir como base para um futuro catálogo web.

---

## 🚀 Como usar um tema

1. Navegue até a pasta `themes/` e escolha um que você goste.
2. Baixe o arquivo `.nlt` correspondente.
3. No seu Android, abra as **Configurações do Niagara** > **Aparência** > **Temas**.
4. Selecione **Community Themes** (Temas da Comunidade) > **Import Community Theme**.
5. Selecione o arquivo `.nlt` que você baixou e pronto!

---

## 🎨 Como contribuir

Adoramos novos temas! Para enviar o seu, siga estes passos:

1. Faça um **Fork** deste repositório.
2. Crie uma pasta dentro de `themes/` com o nome do seu tema (ex: `meu-tema-legal`).
3. Dentro dela, coloque:
   - O arquivo `theme.nlt` (exportado pelo Niagara).
   - Uma imagem `preview.webp` (print da sua tela).
4. (Opcional) Adicione um `info.json` com os créditos.
5. Abra um **Pull Request**.

---

## 🛠️ Estrutura do Repositório

| Arquivo | Descrição |
| :--- | :--- |
| `.nlt` | Arquivo de configuração do tema (cores, fontes, ícones). |
| `.webp` | Imagem de referência para que os usuários vejam o tema antes de baixar. |
| `info.json` | Metadados sobre o criador e recursos utilizados. |
